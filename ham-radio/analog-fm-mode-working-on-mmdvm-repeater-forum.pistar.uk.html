KJ7IEG wrote:Could you show me how do you configure it to work FM Analog mode?

First FM mode doesnt work with a HOTSPOT board. For a REPEATER install you need to load the FM firmware onto ur MMDVM repeater board. All PROVIDING it supports it of course, as not all of them will.

You'll need to be running the current pistar 4.1.2

Then copy this into the mmdvm.ini file

[FM]
Enable=1
# Callsign=
CallsignSpeed=20
CallsignFrequency=1000
CallsignTime=10
CallsignHoldoff=0
CallsignHighLevel=50
CallsignLowLevel=20
CallsignAtStart=1
CallsignAtEnd=1
CallsignAtLatch=0
RFAck=K
ExtAck=N
AckSpeed=20
AckFrequency=1750
AckMinTime=4
AckDelay=1000
AckLevel=50
# Timeout=180
TimeoutLevel=80
CTCSSFrequency=88.4
CTCSSThreshold=30
CTCSSLevel=20
KerchunkTime=0
HangTime=7
UseCOS=1
COSInvert=0
RFAudioBoost=1
MaxDevLevel=90
ExtAudioBoost=1

From personal experience, have a look at the settings in the mmdvm.ini file. If you don't have a COS connection between the radio and the board, set COS to 0.

Also, start pistar up and have a look through the log when it has had time to boot up and get all the services running. If you see anything involving a NAK relating to FM, then it will be more than likely a firmware issue, being incompatible with the mode.

Copy and paste your logfile, maybe we will be able to see something.
