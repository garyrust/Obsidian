Software

For other uses, see Software (disambiguation).

Software written in the JavaScript language

Software consists of computer programs that instruct the execution of a computer.1(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-pis-p16-1) Software also includes design documents and specifications.

The history of software is closely tied to the development of digital computers in the mid-20th century. Early programs were written in the machine language specific to the hardware. The introduction of high-level programming languages in 1958 allowed for more human-readable instructions, making software development easier and more portable across different computer architectures. Software in a programming language is run through a compiler or interpreter to execute on the architecture's hardware. Over time, software has become complex, owing to developments in networking, operating systems, and databases.

Software can generally be categorized into two main types:

  1. operating systems, which manage hardware resources and provide services for applications
  2. application software, which performs specific tasks for users

The rise of cloud computing has introduced the new software delivery model Software as a Service (SaaS). In SaaS, applications are hosted by a provider and accessed over the Internet.

The process of developing software involves several stages. The stages include software design, programming, testing, release, and maintenance. Software quality assurance and security are critical aspects of software development, as bugs and security vulnerabilities can lead to system failures and security breaches. Additionally, legal issues such as software licenses and intellectual property rights play a significant role in the distribution of software products.

History

Main article: History of software

The integrated circuit is an essential invention to produce modern software systems.2(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEJones201419,_22-2)

The first use of the word software to describe computer programs is credited to mathematician John Wilder Tukey in 1958.3(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETracy20212-3)[[4]](https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-4) The first programmable computers, which appeared at the end of the 1940s,5(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEGabbrielliMartini2023519-5) were programmed in machine language. Machine language is difficult to debug and not portable across different computers.6(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEGabbrielliMartini2023520%E2%80%93521-6) Initially, hardware resources were more expensive than human resources.7(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEGabbrielliMartini2023522-7) As programs became complex, programmer productivity became the bottleneck. The introduction of high-level programming languages in 1958 hid the details of the hardware and expressed the underlying algorithms into the code .8(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEGabbrielliMartini2023521-8)[[9]](https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETracy20211-9) Early languages include Fortran, Lisp, and COBOL.9(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETracy20211-9)

Types

See also: Software categories

A diagram showing how the user interacts with application software on a typical desktop computer. The application software layer interfaces with the operating system, which in turn communicates with the hardware. The arrows indicate information flow.

There are two main types of software:

  • Operating systems are "the layer of software that manages a computer's resources for its users and their applications".10(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEAndersonDahlin20146-10) There are three main purposes that an operating system fulfills:11(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEAndersonDahlin20147-11)
    • Allocating resources between different applications, deciding when they will receive central processing unit (CPU) time or space in memory.11(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEAndersonDahlin20147-11)
    • Providing an interface that abstracts the details of accessing hardware details (like physical memory) to make things easier for programmers.11(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEAndersonDahlin20147-11)[[12]](https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETanenbaumBos20235-12)
    • Offering common services, such as an interface for accessing network and disk devices. This enables an application to be run on different hardware without needing to be rewritten.13(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEAndersonDahlin20147,_9,_13-13)
  • Application software runs on top of the operating system and uses the computer's resources to perform a task.14(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEAndersonDahlin20146%E2%80%937-14) There are many different types of application software because the range of tasks that can be performed with modern computers is so large.15(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEJones2014121-15) Applications account for most software16(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETracy202166-16) and require the environment provided by an operating system, and often other applications, in order to function.17(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETracy202172-17)

Comparison of on-premise hardware and software, infrastructure as a service (IaaS), platform as a service (PaaS), and software as a service (SaaS)

Software can also be categorized by how it is deployed. Traditional applications are purchased with a perpetual license for a specific version of the software, downloaded, and run on hardware belonging to the purchaser.18(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan2022386-18) The rise of the Internet and cloud computing enabled a new model, software as a service (SaaS),19(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTECampbell-KellyGarcia-Swartz2015156%E2%80%93157-19) in which the provider hosts the software (usually built on top of rented infrastructure or platforms)20(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTERosatiLynn202023-20) and provides the use of the software to customers, often in exchange for a subscription fee.18(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan2022386-18) By 2023, SaaS products—which are usually delivered via a web application—had become the primary method that companies deliver applications.21(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEWatt20234-21)

Software development and maintenance

Diagram for a traditional software development life cycle from 1988. The numbers represent the typical cost of each phase.

Software companies aim to deliver a high-quality product on time and under budget. A challenge is that software development effort estimation is often inaccurate.22(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan20227-22) Software development begins by conceiving the project, evaluating its feasibility, analyzing the business requirements, and making a software design.23(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan20225-23)[[24]](https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEDooley20171-24) Most software projects speed up their development by reusing or incorporating existing software, either in the form of commercial off-the-shelf (COTS) or open-source software.25(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan202218,_110%E2%80%93111-25)[[26]](https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETracy202143,_76-26) Software quality assurance is typically a combination of manual code review by other engineers27(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan2022117%E2%80%93118-27) and automated software testing. Due to time constraints, testing cannot cover all aspects of the software's intended functionality, so developers often focus on the most critical functionality.28(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan202254-28) Formal methods are used in some safety-critical systems to prove the correctness of code,29(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan2022267-29) while user acceptance testing helps to ensure that the product meets customer expectations.30(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan202220-30) There are a variety of software development methodologies, which vary from completing all steps in order to concurrent and iterative models.31(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan20229-31) Software development is driven by requirements taken from prospective users, as opposed to maintenance, which is driven by events such as a change request.32(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETripathyNaik201426-32)

Frequently, software is released in an incomplete state when the development team runs out of time or funding.33(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEReifer201222-33) Despite testing and quality assurance, virtually all software contains bugs where the system does not work as intended. Post-release software maintenance is necessary to remediate these bugs when they are found and keep the software working as the environment changes over time.34(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETripathyNaik20144,_27-34) New features are often added after the release. Over time, the level of maintenance becomes increasingly restricted before being cut off entirely when the product is withdrawn from the market.35(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETripathyNaik201489-35) As software ages, it becomes known as legacy software and can remain in use for decades, even if there is no one left who knows how to fix it.36(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETracy20213-36) Over the lifetime of the product, software maintenance is estimated to comprise 75 percent or more of the total development cost.37(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEVarga20186-37)[[38]](https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEUlziit_''et_al.''2015764-38)

Completing a software project involves various forms of expertise, not just in software programmers but also testing, documentation writing, project management, graphic design, user experience, user support, marketing, and fundraising.39(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETuckerMorellide_Silva20117-39)[[40]](https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEStull201824%E2%80%9325-40)[[24]](https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEDooley20171-24)

Quality and security

Main articles: Software quality and Computer security

Software quality is defined as meeting the stated requirements as well as customer expectations.41(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEGalin20183-41) Quality is an overarching term that can refer to a code's correct and efficient behavior, its reusability and portability, or the ease of modification.42(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEGalin201826-42) It is usually more cost-effective to build quality into the product from the beginning rather than try to add it later in the development process.43(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan202268,_117-43) Higher quality code will reduce lifetime cost to both suppliers and customers as it is more reliable and easier to maintain.44(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan20223,_268-44)[[45]](https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEVarga201812-45) Software failures in safety-critical systems can be very serious including death.44(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan20223,_268-44) By some estimates, the cost of poor quality software can be as high as 20 to 40 percent of sales.46(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan2022119-46) Despite developers' goal of delivering a product that works entirely as intended, virtually all software contains bugs.47(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEAblonBogart20171-47)

The rise of the Internet also greatly increased the need for computer security as it enabled malicious actors to conduct cyberattacks remotely.48(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTECampbell-KellyGarcia-Swartz2015164-48)[[49]](https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan2022266-49) If a bug creates a security risk, it is called a vulnerability.50(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEAblonBogart20172-50)[[51]](https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEDaswaniElbayadi202125-51) Software patches are often released to fix identified vulnerabilities, but those that remain unknown (zero days) as well as those that have not been patched are still liable for exploitation.52(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEDaswaniElbayadi202126%E2%80%9327-52) Vulnerabilities vary in their ability to be exploited by malicious actors,50(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEAblonBogart20172-50) and the actual risk is dependent on the nature of the vulnerability as well as the value of the surrounding system.53(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEHaberHibbert20185%E2%80%936-53) Although some vulnerabilities can only be used for denial of service attacks that compromise a system's availability, others allow the attacker to inject and run their own code (called malware), without the user being aware of it.50(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEAblonBogart20172-50) To thwart cyberattacks, all software in the system must be designed to withstand and recover from external attack.49(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan2022266-49) Despite efforts to ensure security, a significant fraction of computers are infected with malware.54(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEKitchinDodge201137-54)

Encoding and execution

Programming languages

Main article: Programming language

The source code for a computer program in C. The gray lines are comments that explain the program to humans. When compiled and run, it will output "Hello, world!".

Programming languages are the format in which software is written. Since the 1950s, thousands of different programming languages have been invented; some have been in use for decades, while others have fallen into disuse.55(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETracy2021117-55) Some definitions classify machine code—the exact instructions directly implemented by the hardware—and assembly language—a more human-readable alternative to machine code whose statements can be translated one-to-one into machine code—as programming languages.56(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETracy2021118%E2%80%93120-56) Programs written in the high-level programming languages used to create software share a few main characteristics: knowledge of machine code is not necessary to write them, they can be ported to other computer systems, and they are more concise and human-readable than machine code.57(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETracy2021118%E2%80%93119-57) They must be both human-readable and capable of being translated into unambiguous instructions for computer hardware.58(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEKitchinDodge201126-58)

Compilation, interpretation, and execution

The invention of high-level programming languages was simultaneous with the compilers needed to translate them automatically into machine code.59(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETracy2021121-59) Most programs do not contain all the resources needed to run them and rely on external libraries. Part of the compiler's function is to link these files in such a way that the program can be executed by the hardware. Once compiled, the program can be saved as an object file and the loader (part of the operating system) can take this saved file and execute it as a process on the computer hardware.60(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTETracy2021122%E2%80%93123-60) Some programming languages use an interpreter instead of a compiler. An interpreter converts the program into machine code at run time, which makes them 10 to 100 times slower than compiled programming languages.61(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan2022375-61)[[62]](https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTESebesta201228-62)

Legal issues

Liability

Main article: Software product liability

Software is often released with the knowledge that it is incomplete or contains bugs. Purchasers knowingly buy it in this state, which has led to a legal regime where liability for software products is significantly curtailed compared to other products.63(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEKitchinDodge201136%E2%80%9337-63)

Licenses

Main articles: Software license and Software copyright

Blender, a free software program

Since the mid-1970s, software and its source code have been protected by copyright law that vests the owner with the exclusive right to copy the code. The underlying ideas or algorithms are not protected by copyright law, but are sometimes treated as a trade secret and concealed by such methods as non-disclosure agreements.64(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan2022394%E2%80%93396-64) A software copyright is often owned by the person or company that financed or made the software (depending on their contracts with employees or contractors who helped to write it).65(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan2022403-65) Some software is in the public domain and has no restrictions on who can use it, copy or share it, or modify it; a notable example is software written by the United States Government. Free and open-source software also allow free use, sharing, and modification, perhaps with a few specified conditions.65(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan2022403-65) The use of some software is governed by an agreement (software license) written by the copyright holder and imposed on the user. Proprietary software is usually sold under a restrictive license that limits its use and sharing.66(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan2022394,_404-66) Some free software licenses require that modified versions must be released under the same license, which prevents the software from being sold or distributed under proprietary restrictions.67(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTELanger201644%E2%80%9345-67)

Patents

Main articles: Software patent and Software patent debate

Patents give an inventor an exclusive, time-limited license for a novel product or process.68(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan2022395-68) Ideas about what software could accomplish are not protected by law and concrete implementations are instead covered by copyright law. In some countries, a requirement for the claimed invention to have an effect on the physical world may also be part of the requirements for a software patent to be held valid.69(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-69) Software patents have been historically controversial. Before the 1998 case [State Street Bank & Trust Co. v. Signature Financial Group, Inc.](https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/State_Street_Bank%26Trust_Co._v._Signature_Financial_Group%2C_Inc. "State Street Bank & Trust Co. v. Signature Financial Group, Inc."), software patents were generally not recognized in the United States. In that case, the Supreme Court decided that business processes could be patented.70(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEJones201419-70) Patent applications are complex and costly, and lawsuits involving patents can drive up the cost of products.71(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan2022398-71) Unlike copyrights, patents generally only apply in the jurisdiction where they were issued.72(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEO'Regan2022399-72)

Impact

Further information: Information Age

Computer-generated simulations are one of the advances enabled by software.73(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEManovich2013333-73)

Engineer Capers Jones writes that "computers and software are making profound changes to every aspect of human life: education, work, warfare, entertainment, medicine, law, and everything else".74(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEJones201432-74) It has become ubiquitous in everyday life in developed countries.75(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEKitchinDodge2011iv-75) In many cases, software augments the functionality of existing technologies such as household appliances and elevators.76(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEKitchinDodge20115-76) Software also spawned entirely new technologies such as the Internet, video games, mobile phones, and GPS.76(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEKitchinDodge20115-76)[[77]](https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEJones2014xxviii-77) New methods of communication, including email, forums, blogs, microblogging, wikis, and social media, were enabled by the Internet.78(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEManovich2013329-78) Massive amounts of knowledge exceeding any paper-based library are now available with a quick web search.77(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEJones2014xxviii-77) Most creative professionals have switched to software-based tools such as computer-aided design, 3D modeling, digital image editing, and computer animation.73(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEManovich2013333-73) Almost every complex device is controlled by software.77(https://browse.library.kiwix.org/viewer#wikipedia_en_all_maxi_2025-08/Software#cite_note-FOOTNOTEJones2014xxviii-77)
