🦀 ZeroClaw on Raspberry Pi Zero 2 W — Static Rust Binary + systemd (Secure Install)

This video shows a fully static, systemd-managed ZeroClaw deployment on a Raspberry Pi Zero 2 W. No Node. No npm. No runtime dependencies. No shell installer. Just one statically linked ARM64 binary, one locked-down service user, and one canonical execution path. This is closer to how embedded appliances are deployed — not how developer tools are usually installed. We build the binary properly on macOS using Rust + MUSL, cross-compile for aarch64, transfer it cleanly, and deploy it as a hardened systemd service. This is not a “quick start.” This is an appliance-grade install.
