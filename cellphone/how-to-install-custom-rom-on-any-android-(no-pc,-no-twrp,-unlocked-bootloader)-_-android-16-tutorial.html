Install Custom ROM on Any Android Phone in 2025 — even if your device has no official ROM on XDA. In this video, I’ll show you how I turned an old Moto phone stuck on Android 12 into a device running Android 16 with all the new features, using a GSI ROM and without using a PC.

This guide works on all Android phones that support bootloader unlocking, including OnePlus, Samsung, Nothing, Motorola, Realme, Xiaomi, Pixel and more — as long as the device supports Project Treble.

If your phone doesn’t have a dedicated custom ROM available, you can still install a GSI ROM like Evolution X GSI, LineageOS GSI, and many others. This video shows the full process step by step.
